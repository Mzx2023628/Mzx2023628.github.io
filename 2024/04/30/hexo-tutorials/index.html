hexo建站攻略

安装git

  • 下载地址：
    https://git-scm.com/download/win

安装node.js

  • 下载地址：https://nodejs.org/en

安装hexo

  • 在任意位置右键，选择 Open Git Bash Here ，然后输入

    npm install -g hexo-cli

  • 回车之后，开始安装hexo

创建文件夹

  • 在C:\用户\ZK目录下创建 Mzx’s Blog 的文件夹。

初始化文件夹

  • 进入创建的文件夹，右键，选择 Open Git Bash Here ，然后输入

    hexo init

安装依赖包

  • 在 Git Bash 中输入
    • npm install

本地运行hexo

  • 在 Git Bash 中输入
    • hexo generate
    • hexo server
  • 完成后在浏览器中打开：http://localhost:4000
    这就是本地的hexo，每次提交文章都可以现在本地看一眼，再去提交

创建仓库

  • 在GitHub上创建博客的仓库，GitHub地址：https://github.com

设置SSH keys

  • 生成SHH keys， 在Git Bash中输入下面的命令，查看是否生成过 .ssh文件, 如果有，删除
    ls -al ~/.ssh
  • 然后依次输入下面的命令，邮箱为你自己申请github的邮箱
    ssh-keygen -t rsa -C “example@qq.com“
    ssh-agent -s
    ssh-add ~/.ssh/id_rsa
  • 如果出现Could not open a connection to your authentication agent.则依次输入下面的指令
  • 点击github的头像，选择Settings，然后依次点击左侧的SSHand GPG keys和右上角的New SSH key
    title取名随意，复制公钥到key的文本框中，点击Add SHH key，公钥如下
    cat ../.ssh/id_rsa.pub

测试

  • 在Git Bash中输入
    ssh -T git@github.com
    出现如下表示测试成功，SSH配置成功

部署到GitHub上

  • 在Git Bash中输入
    hexo generate
    hexo deploy
  • 如果出现以下错误
    ERROR Deployer not found: github
    需要安装hexo-deployer-git模块，在Git Bash中输入
    npm install hexo-deployer-git –save
    安装好之后重新执行
    hexo deploy

访问博客

  • 在浏览器打开：https://mzx2023628.github.io/ ，即可访问基于hexo的博客了！
    远程部署会有延时，耐心等待几分钟即可

常用命令

  • hexo clean
  • hexo g
  • hexo s
  • hexo d
  • 三部曲：写好博客以后，首先保存，在Git Bash中输入 hexo clean ，再输入 hexo g ，再输入hexo s ，可以在本地 http://localhost:4000/ 中先查看，确认无误，再上传远程 hexo d 。
