Markdown tutorials

发表于 2024-04-29 更新于 2024-04-30 分类于 Markdown

注：VScode 在命令窗口输入 Markdown: Open Preview to the Side 命令， 可以进行实时预览

Markdown标题

使用#号可以表示1-6级标题，一级标题对应一个#号，二级标题对应两个#号，以此类推。

Markdown段落

段落换行使用两个以上空格加上回车，或者可以在段落后面使用一个空行来表示重新开始一个段落

Markdown字体

斜体文本，文本两边加一个*
粗体文本，文本两边加两个*
粗斜体文本，文本两边加三个*

Markdown分隔线

可以在一行中使用三个以上的号来建立一个分隔线，行内不能有其他东西，也可以在号中间插入空格

Markdown删除线

只需要在文本的两端加两个~号即可
删除线

Markdown下划线

通过在文本两边加实现
下划线

Markdown脚注

脚注是对文本的补充说明
创建脚注格式[^1]。
[^1]:脚注

Markdown列表

无序列表，使用号作为列表标记，号后面要加空格。

  • 第一项
  • 第二项
  • 第三项

有序列表使用数字加上.号来表示，也要加空格

  1. 第一项
  2. 第二项
  3. 第三项

列表嵌套，只需在子列表中的选项前面添加四个空格即可

  1. 第一项
    • 元素一
    • 元素二
  2. 第二项
    • 元素一
    • 元素二

Markdown区块

Markdown 区块引用是在段落开头使用 > 符号 ，然后后面紧跟一个空格符号

区块
区块
区块

区块是可以嵌套的，一个 > 符号是最外层，两个 > 符号是第一层嵌套，以此类推

Markdown代码

如果是段落上的一个函数或片段的代码可以用`号包起来

printf()函数

代码区块
代码区块使用 4 个空格或者一个制表符（Tab 键） 

int main()
{
    printf("hello world!");
    return 0;
}

也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）

Markdown链接

[]里填链接名称，()里填链接地址或者()换<>
这是一个链接—百度

高级链接，可以通过变量来设置一个链接，变量赋值在文档末尾进行

链接[百度][1]
链接[4399][2]
文档结尾为变量赋值
[1]:https://www.baidu.com
[2]:https://www.4399.com

Markdown图片

Markdown 图片语法格式如下
开头一个感叹号 !
接着一个方括号，里面放上图片的替代文字
接着一个普通括号，里面放上图片的网址，最后还可以用引号包住并加上选择性的 ‘title’ 属性的文字

也可以像网址那样对图片网址使用变量

Markdown表格

制作表格使用 | 来分隔不同的单元格，使用 - 来分隔表头和其他行
